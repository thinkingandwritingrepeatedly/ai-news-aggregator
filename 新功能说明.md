## 📋 快速配置（3 步）

### 第 1 步：获取凭证

**QQ 邮箱授权码：**
1. 登录 QQ 邮箱 (https://mail.qq.com)
2. 设置 → 账户 → POP3/IMAP/SMTP
3. 开启服务，获取授权码

**Anthropic API Key：**
1. 访问 https://console.anthropic.com
2. 创建 API Key

### 第 2 步：配置凭证

编辑 `daily-run.bat`，修改以下行：

```batch
set EMAIL_USER=your_qq_email@qq.com
set EMAIL_PASSWORD=your_qq_email_auth_code
set ANTHROPIC_API_KEY=your_anthropic_api_key
```

### 第 3 步：设置定时任务

1. 打开 "任务计划程序"
2. 创建基本任务
3. 名称：`AI News Daily Report`
4. 触发器：每天 09:00
5. 操作：运行 `daily-run.bat`
