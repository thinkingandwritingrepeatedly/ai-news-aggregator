# 快速参考

## 运行命令

### 基础版（无需 API key）
```bash
# Windows
run.bat

# Linux/Mac
node node_modules/tsx/dist/cli.mjs src/index.ts
```

### AI 版（需要 API key）
```bash
# Windows - 编辑 run-with-ai.bat，然后双击运行
# 或在 PowerShell 中：
$env:ANTHROPIC_API_KEY = "sk-ant-xxxxxxxxxxxxx"
node node_modules/tsx/dist/cli.mjs src/index.ts

# Linux/Mac
export ANTHROPIC_API_KEY="sk-ant-xxxxxxxxxxxxx"
node node_modules/tsx/dist/cli.mjs src/index.ts
```

## 配置修改

### 修改新闻源
编辑 `src/index.ts`：
```typescript
const FEED_SOURCES: FeedSource[] = [
  { name: '新源', url: 'https://...', type: 'rss' },
];
```

### 修改摘要长度
编辑 `src/summarizer.ts`：
```typescript
content: `请用一句话（不超过50个字）...`  // 改为你想要的字数
```

### 修改使用的模型
编辑 `src/summarizer.ts`：
```typescript
model: 'claude-opus-4-7',  // 改为 'claude-sonnet-4-6' 或 'claude-haiku-4-5'
```

### 修改输出目录
编辑 `src/index.ts`：
```typescript
const outputDir = join(__dirname, '..', 'output');  // 改为你想要的路径
```

## 常见错误

| 错误 | 原因 | 解决方案 |
|------|------|--------|
| `ANTHROPIC_API_KEY not set` | 未设置 API key | 设置环境变量或使用基础版 |
| `fetch failed` | 网络连接问题 | 检查网络，重试 |
| `HTTP 502` | RSS 源服务器错误 | 等待服务器恢复 |
| `Translation failed` | 翻译 API 失败 | 自动使用本地字典 |
| `Failed to generate summary` | API 调用失败 | 检查 API key，重试 |
