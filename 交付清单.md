## 🚀 使用方式

### 快速开始

```bash
# 基础版（无需 API key）
npm install
npm start

# AI 版（需要 API key）
export ANTHROPIC_API_KEY="sk-ant-xxxxxxxxxxxxx"
npm start
```

### 输出

日报保存在 `output/ai-news-YYYY-MM-DD.md`
