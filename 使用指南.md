# 使用指南

## 快速开始

### 1. 基础运行（不需要 API key）

```bash
# Windows - 双击运行
run.bat

# 或在命令行运行
node node_modules/tsx/dist/cli.mjs src/index.ts
```

**功能：**
- ✅ 抓取最新 AI 新闻
- ✅ 过滤 24 小时内的文章
- ✅ 去重处理
- ✅ 本地字典翻译摘要
- ❌ 不使用 AI 总结（摘要是原始描述）

---

### 2. 高级运行（使用 Claude API 生成 AI 总结）

#### 第一步：获取 API Key

1. 访问 [Anthropic Console](https://console.anthropic.com)
2. 登录或注册账户
3. 创建 API key
4. 复制 API key

#### 第二步：设置环境变量

**方式 A：编辑 run-with-ai.bat（Windows）**

```batch
@echo off
REM 将 your_api_key_here 替换为你的实际 API key
set ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxx

cd /d "%~dp0"
"C:\Program Files\nodejs\node.exe" node_modules/tsx/dist/cli.mjs src/index.ts
pause
```

然后双击运行 `run-with-ai.bat`

**方式 B：命令行运行（Windows PowerShell）**

```powershell
$env:ANTHROPIC_API_KEY = "sk-ant-xxxxxxxxxxxxx"
node node_modules/tsx/dist/cli.mjs src/index.ts
```

**方式 C：命令行运行（Linux/Mac）**

```bash
export ANTHROPIC_API_KEY="sk-ant-xxxxxxxxxxxxx"
node node_modules/tsx/dist/cli.mjs src/index.ts
```

#### 第三步：运行程序

程序会自动：
1. 检测 API key
2. 抓取新闻
3. **使用 Claude 生成一句话总结**（最多 50 字）
4. 翻译摘要为中文
5. 生成日报

**输出示例：**

```
🚀 Starting AI news aggregation...

📡 Fetching feeds...
✓ Fetched 60 articles

🔍 Processing articles...
✓ Filtered to 45 articles from last 24 hours
✓ Deduplicated to 43 unique articles
✓ Found articles from 3 sources

🤖 Generating AI summaries...
✓ Summarization completed

🌐 Translating summaries to Chinese...
✓ Translation completed

📝 Generating markdown...
✓ Report saved to: output/ai-news-2026-05-28.md

✅ Done!
```

## 常见问题

### Q1: 没有 API key 可以用吗？

**A:** 可以！基础版本不需要 API key，但摘要质量会较低（直接使用原始描述）。

### Q2: API key 安全吗？

**A:** 
- ✅ API key 只在本地使用，不会上传到任何地方
- ✅ 建议不要把 API key 提交到 Git
- ✅ 可以在 `.gitignore` 中添加 `.env` 文件

### Q3: 使用 API 会产生费用吗？

**A:** 是的，每次调用 Claude API 都会产生费用。费用取决于：
- 输入 token 数量（标题 + 描述）
- 输出 token 数量（生成的总结）
- 使用的模型（Opus 4.7 比较贵）

**成本估算：**
- 每篇文章约 0.01-0.05 美元
- 50 篇文章约 0.50-2.50 美元

### Q4: 如何降低成本？

**A:** 可以修改 `src/summarizer.ts` 中的模型：

```typescript
// 改为更便宜的模型
model: 'claude-opus-4-7',  // 改为 'claude-sonnet-4-6' 或 'claude-haiku-4-5'
```

### Q5: 总结不满意怎么办？

**A:** 可以修改 `src/summarizer.ts` 中的 prompt：

```typescript
content: `请用一句话（不超过50个字）总结这篇文章的核心内容。
...`
```

改为你想要的风格，比如：
- "用专业术语总结"
- "用通俗语言总结"
- "突出创新点"

### Q6: 如何定时运行？

**A:** 使用系统任务计划程序

**Windows：**
1. 打开"任务计划程序"
2. 创建基本任务
3. 设置触发器（每天 8 点）
4. 设置操作：运行 `run-with-ai.bat`

**Linux/Mac：**
```bash
# 编辑 crontab
crontab -e

# 每天早上 8 点运行
0 8 * * * export ANTHROPIC_API_KEY="sk-ant-xxx" && cd /path/to/project && node node_modules/tsx/dist/cli.mjs src/index.ts
```

### Q7: 如何修改新闻源？

**A:** 编辑 `src/index.ts` 中的 `FEED_SOURCES`：

```typescript
const FEED_SOURCES: FeedSource[] = [
  {
    name: '新源名称',
    url: 'https://example.com/feed.xml',
    type: 'rss',
  },
];
```

### Q8: 如何修改摘要长度？

**A:** 编辑 `src/summarizer.ts`：

```typescript
content: `请用一句话（不超过50个字）...`  // 改为你想要的字数
```

## 故障排除

### 问题：API key 无效

**解决方案：**
1. 检查 API key 是否正确复制
2. 确保 API key 没有过期
3. 检查 API key 是否有足够的配额

### 问题：网络连接失败

**解决方案：**
1. 检查网络连接
2. 检查防火墙设置
3. 尝试使用代理

### 问题：生成速度很慢

**解决方案：**
1. 这是正常的（调用 API 需要时间）
2. 可以改用更快的模型（Haiku）
3. 可以减少文章数量

### 问题：摘要是英文而不是中文

**解决方案：**
1. 这是正常的（AI 总结是英文）
2. 翻译模块会自动翻译
3. 如果翻译失败，会显示英文
