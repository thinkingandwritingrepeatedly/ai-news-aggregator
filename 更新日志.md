# 更新日志

## 2026-05-28 更新

### 新增功能：AI 驱动的摘要生成

#### 功能说明
- 使用 Claude API 为每篇文章生成高质量的一句话总结
- 摘要长度限制在 50 字以内
- 自动从标题 + 原始描述生成总结
- 支持可选的 API key 配置

#### 工作原理
```
原始数据：
  标题：ElevenLabs's new music generation model can switch genres mid-track
  描述：ElevenLabs' new model will let users regenerate a section of a song...

↓ Claude API 处理

生成的总结：
  ElevenLabs 发布新的音乐生成模型，支持在播放过程中切换音乐风格。
```

#### 使用方式

**基础版（不需要 API key）：**
```bash
node node_modules/tsx/dist/cli.mjs src/index.ts
```

**AI 版（需要 API key）：**
```bash
set ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxx
node node_modules/tsx/dist/cli.mjs src/index.ts
```

#### 新增文件
- `src/summarizer.ts` - AI 摘要生成模块
- `使用指南.md` - 详细的使用说明
- `run-with-ai.bat` - Windows 快捷运行脚本
- `run-with-api-key.sh` - Linux/Mac 快捷运行脚本
- `.env.example` - 环境变量示例

#### 改进点
1. **摘要质量提升** - 从简单截取改为 AI 生成
2. **灵活配置** - 支持有无 API key 两种模式
3. **错误处理** - API 调用失败时自动降级
4. **成本控制** - 可配置模型选择（Opus/Sonnet/Haiku）

#### 成本估算
- Opus 4.7：每篇文章 0.01-0.05 美元
- Sonnet 4.6：每篇文章 0.003-0.01 美元
- Haiku 4.5：每篇文章 0.0005-0.002 美元

#### 配置选项

修改 `src/summarizer.ts` 中的模型：
```typescript
// 改为更便宜的模型
model: 'claude-sonnet-4-6',  // 或 'claude-haiku-4-5'
```

修改摘要长度：
```typescript
content: `请用一句话（不超过100个字）...`  // 改为你想要的字数
```
